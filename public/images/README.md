# Logo Setup Instructions

Please save the uploaded "Resume Unleashed" logo image as:
- `/public/images/logo.png` - For use in navbar, footer, and sidebar

The logo should be the white text on black background image provided by the user.

For the favicon, you can use an online tool to convert the logo to .ico format and save it as:
- `/src/app/favicon.ico` (replace existing)

Alternatively, you can use Next.js Image Optimization by saving as:
- `/public/favicon.ico`
- `/public/apple-touch-icon.png`
- `/public/favicon-16x16.png`
- `/public/favicon-32x32.png`
